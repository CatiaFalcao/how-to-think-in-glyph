## ğŸ§© What is GLYPH?

**GLYPH** is a universal language for structured problem formulation.

It provides:
- a canonical structure for describing problems
- strict separation between observation, interpretation, and action
- a validation framework for problem clarity
- a foundation for reusable, comparable problem knowledge

GLYPH is **not** a solution framework.  
It is a **thinking framework**.
